# JogoTeste

Por enquanto um jogo sem nome, feito como forma de aprender conceitos de C# e Windows forms, além de utilizar conceitos de Game Design

A ideia inicial é que seja um Roguelike de Turnos, com mecânicas interativas que mudam a gameplay, manipulando a taxa de acerto por exemplo

Futuramente as ideias vão ser mais refinadas, e o jogo vai ficar mais bonito etc, quando for no mínimo jogável esse readme vai ser atualizado explicando
